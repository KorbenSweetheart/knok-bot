# KNOK Bot or Knowledge Keeper bot

The bot saves articles for you to read later.

It can be useful for people who often save many links, but always forget about them.

## Usage:

Send a link to save an article.

```
https://website.com/url-path/
```

### Commands:

- `/start` - to launch the bot
- `/help` - display help message
- `/rnd` - get random page. ⚠️ Page will be removed from the list
